## 介绍

用于实现 KPlayer 推流过程中在指定位置显示图片的插件，支持透明通道PNG图片。可用于显示水印、LOGO、遮挡功能

## 配置文件

```json
{
  "plugin": {
    "lists": [
      {
        "path": "show-image",
        "unique": "my_plugin",
        "params": {
          "filename": "my.png"
        }
      }
    ]
  }
}
```



## 参数列表

| 参数     | 说明                                               | 默认值      |
| -------- | -------------------------------------------------- | ----------- |
| filename | 图片文件路径                                       | example.png |
| x        | 以视频左上角为原点的横坐标，最大可视范围为分辨率宽 | 0           |
| y        | 以视频左上角为原点的纵坐标，最大可视范围为分辨率高 | 0           |
